1. position: relative vs absolute

position: relative:

  • Vẫn nằm trong luồng (flow) bình thường
  • Dịch chuyển bằng top, left nhưng giữ vị trí gốc

position: absolute:

  • Bị loại khỏi flow
  • Định vị theo phần tử cha có position

Ứng dụng:

  • relative: làm mốc cho absolute
  • absolute: overlay, badge, text trên ảnh

Responsive:

  • absolute dễ vỡ layout nếu dùng px cố định
  • Nên dùng %, vw, vh và kết hợp Flexbox/Grid

Deploy:

  • Không ảnh hưởng trực tiếp bởi hosting
  • Nhưng dễ lỗi trên mobile nếu dùng sai

2. Domain, Hosting và DNS

Quy trình:

  • Mua domain
  • Deploy website
  • Cấu hình DNS (A, CNAME)

DNS:

  • Chuyển domain → IP
  • Kết nối người dùng với server

Công cụ deploy:

  • GitHub Pages
  • Netlify
  • Vercel

Rủi ro:

  • Domain takeover
  • Không có HTTPS
  • Lộ API key

3. var, let, const

Scope:

  • var: function scope
  • let, const: block scope

Hoisting:

  • var: undefined
  • let, const: lỗi (TDZ)

Lỗi thường gặp:

for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}
// Kết quả: 3 3 3

Fix:

for (let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}
// Kết quả: 0 1 2

Debug:

  • Dùng DevTools (F12)
  • Đặt breakpoint
  • Quan sát scope

4. Event Delegation

Khái niệm:

  • Gắn event vào phần tử cha
  • Dùng event bubbling để xử lý con

Ví dụ:

list.addEventListener('click', (e) => {
  if (e.target.classList.contains('item')) {
    console.log('Clicked');
  }
});

Lợi ích:

  • Giảm số lượng event listener
  • Hỗ trợ dynamic content
  • Tăng hiệu suất

Kết hợp HTML/CSS:

  • Dùng class để xử lý hành vi
  • Dùng data-* để lưu dữ liệu
